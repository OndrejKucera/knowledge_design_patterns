## Null Object

To avoid scattering null checks throughout our code by encapsulating the action
 - lack of a value in Java is represented by a null reference
 
### Java Example
 ```java
 if(null == someObject){
   // default null handling behavior
 } else {
   someObject.someMethod()
 }
 ```
